Задача №3 Position Tracking evaluation

Входные данные:
Два файла с координатами перемещения устройства в пространстве:
- gt.txt (эталонные данные позиционирования, созданы синтетически)
- est.txt (данные тестируемого алгоритма позиционирования)

Задача:
1) Оценить точность позиционирования тестируемого алгоритма относительно 
эталонных данных. Для оценки необходимо выполнить подсчет метрик:
- Drift Per Second
- Final Drift
- Max Jittering
- Average Jittering
- Total Distance

* Перед выполнением подсчета рекомендуется свести две траектории по параметру "timestamp" и удалить промежуточные координаты из файла gt.txt

2) Визуализировать две траектории, для наглядной демонстрации возможных ошибок позиционирования тестируемого алгоритма.

3) Создать отчет в свободной форме.
==========================================================

Объяснение метрик:
- Drift - aбсолютное отклонение точек, разница приращений каждой точки с первой между измеряемой и эталонной траекторией
- Avg Drift - среднее значение Drift
- Drift Per Second - Avg Drift за время траектории
- Final Drift - разница приращений последней точки с первой между измеряемой и эталонной траекторией
-------------
- Jittering - относительное отклонение точек, разница приращений каждой точки с предыдущей между измеряемой и эталонной траекторией
- Max Jittering - максимальное значение Jittering
- Average Jittering - среднее значение Jittering
-------------
- Total Distance - сумма расстояний между соседними точками на всей траектории (как для измеряемой так и для эталонной траектории )
