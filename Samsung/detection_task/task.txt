Задача №2 Object detection evaluation

Входные данные:
Две базы с данными детектирования объектов на 2D изображении:
groundtruth.h5 - База данные с эталонными детектами
predicted.h5 - База данных с детектами тестируемого алгоритма

Задача:
1) Оценить точность детектирования объектов типа "Pedestrian" 
относительно эталонных данных. Для оценки необходимо выполнить подсчет метрик:
- Precision 
- Recall

2) Визуализировать все центры детектов класса "Pedestrian" на 
одном изображении (heat map), для визуального анализа плотности всех 
детектов тестируемого алгоритма относительно эталонных данных.
- для groundtruth и для predicted отдельно

3) Создать отчет в свободной форме.
